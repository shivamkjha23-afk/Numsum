## Compatibility redirects

- `/competitions` → `/challenges`
- `/competitions/[id]` → `/challenges/[id]`
- `/competitions/[id]/join` → `/challenges/[id]/participate`
- `/admin/competitions` → `/admin/challenges`
- `/admin/competitions/[id]` → `/admin/challenges/[id]`
- `/dashboard/competitions` → `/dashboard/challenges`
- `/dashboard/competitions/teams` → `/dashboard/challenges`
- `/dashboard/competitions/submissions` → `/dashboard/challenges`
